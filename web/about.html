Spiral

Written in Kotlin, Spiral aims to be a multi-platform manager for mods for Danganronpa. By leveraging the game engine itself, Spiral enables authors to write content that looks authentic, because it is. Your fanproject can be run on the same base engine that powers any of the titles supported by Spiral.

While creating any kind of meaningful content using the raw formats would be a high level of tedium, Spiral aims to improve this experience by providing easy-to-use interfaces, intuitive scripting, and standard, everyday formats for formats such as video, audio, and images.

Progress

Work on Spiral continues every day, and progress continues to be made. The console version is currently being rewritten to facilitate the new features that are required since the original Kotlin version was written.
